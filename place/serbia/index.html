34 Legislation
URL: http://www.parlament.gov.rs/akti/doneti-zakoni/doneti-zakoni.1033.html
Format: PDF, DOC

All national legislation enacted since 2001 can be retrieved by metadata and browsed by volume and number. The data is not machine readable (it is available in PDF and DOC format). No licence is visible (although the legislation should be in the public domain).

More comprehensive legislation set is available at http://pravno-informacioni-sistem.rs and http://bpp.uzzpro.gov.rs, but it also isn't machine readable.

16 Government Spending


As part of the budget, it is possible to find data on total expenditure and expenditure for the current year. Data are related to individual sectors. There is no detailed transactions on a monthly basis. http://www.trezor.gov.rs/uploads/file/Zakoni/Zakon%20o%20budzetu%20za%202013.pdf

Given there are no detailed monthly transactions available this dataset needs to be marked as not publicly available (but the data seems to be available and digital which is needed to produce the aggregated data).
